# Churn Prediction – IA + Negocio

Proyecto de análisis y predicción de churn (abandono de clientes) usando
Machine Learning y enfoque de negocio.

# Objetivo
Predecir la probabilidad de que un cliente abandone el servicio y clasificar
su nivel de riesgo para apoyar estrategias de retención.

# Metodología
- Análisis exploratorio de datos (EDA)
- Limpieza y preprocesamiento
- Pipeline con:
  - Escalado de variables numéricas
  - One-Hot Encoding para variables categóricas
  - Regresión Logística
- Evaluación con:
  - Matriz de confusión
  - Precision, Recall y F1-score
  - Probabilidad de churn por cliente

# Tecnologías
- Python
- Pandas / NumPy
- Scikit-learn
- Jupyter Notebook

# Resultados
- Accuracy ≈ 86%
- Identificación de clientes con alto, medio y bajo riesgo de churn
